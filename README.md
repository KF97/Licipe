# 들어주는 레시피 서비스 : Licipe

## :monocle_face: 개요
  요리 중, 손을 이용한 조작이 필요한 기존의 레시피 서비스들로 인해 지저분해지는 스크린과 터치 인식 오류 문제 등을 해결하기 위해 
  개발한 음성 인식 기반 레시피 서비스입니다.
